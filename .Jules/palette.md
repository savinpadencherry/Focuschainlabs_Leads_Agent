## 2026-06-13 - Visually Hidden Meaningful Buttons\n**Learning:** Overlay buttons with empty strings (e.g., `" "`) as text combined with `opacity: 0` create a poor experience for screen reader users, since the button label is empty. \n**Action:** Use a meaningful label like `"View details"` and rely on CSS to hide it visually, and add a `help="..."` parameter so sighted users get a tooltip on hover.
